# Neofy - Regenerative Infrastructure for Circular Opportunity

A professional, clean, and modern website for Neofy, showcasing the company's innovative Black Soldier Fly bioconversion technology and circular economy solutions across East Africa.

## 🌟 Features

- **Clean, Professional Design**: Modern UI with smooth green color palette
- **Mobile-First Responsive**: Optimized for all devices with compact, efficient layouts
- **Interactive Elements**: Smooth animations, form handling, and user feedback
- **Performance Optimized**: Fast loading with efficient CSS and JavaScript
- **Accessibility Focused**: Keyboard navigation, focus management, and semantic HTML
- **SEO Ready**: Proper meta tags, semantic structure, and optimized content

## 🚀 Technology Stack

- **HTML5**: Semantic markup with proper structure
- **CSS3**: Modern styling with CSS Grid, Flexbox, and custom properties
- **Vanilla JavaScript**: Lightweight, performant interactions
- **Google Fonts**: Inter font family for clean typography
- **No Dependencies**: Pure vanilla code for maximum performance
